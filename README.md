## 📋 Sobre o Projeto
Este é um sistema de classificação de heróis desenvolvido para consolidar fundamentos de lógica de programação. O algoritmo processa o nome, a classe e a experiência (XP) de um personagem para determinar seu nível hierárquico, variando de **Ferro** até o lendário nível **Radiante**.

## 🛠️ Tecnologias
- **JavaScript (ES6+)**: Utilizado para toda a lógica condicional do sistema.
- **Node.js**: Ambiente de execução para rodar o script via terminal.

## 🧠 Conceitos Aplicados
Como uma desenvolvedora perfeccionista, foquei em aplicar:
- **Arrays**: Para armazenamento estruturado dos dados da heroína.
- **Estruturas de Decisão (If/Else If)**: Para garantir que cada faixa de XP fosse validada com precisão.
- **Concatenação de Strings**: Para uma saída de dados limpa e amigável no console.
